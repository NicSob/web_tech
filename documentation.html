To list the data of a specific item - (Retrieve)

endpoint URI: /author/:id

MIME format: application/json

HTTP method: GET

Example Input:

 GET /author/1

Example Output:

        Status: OK
        {
            "id": 1,
            "author": "Tim Berners-Lee",
            "alt": "Image of Berners-Lee",
            "tags": "html,http,url,cern,mit",
            "image": "https://upload.wikimedia.org/wikipedia/commons/9/9d/Sir_Tim_Berners-Lee.jpg",
            "description": "The internet and the Web aren't the same thing."
        }

To change data of a specific item - (Update)

endpoint URI: /author/:id

MIME format: application/json

HTTP method: PUT

Example Input:

 PUT /author/3
        {
            "author": "Isaac Newton"   
        }

Example Output:

        204 No Content
        Status: OK
        Row(s) updated: 1

To remove data of a specific item - (Delete)

endpoint URI: /author/:id

MIME format: application/json

HTTP method: DELETE

Example Input:

 DELETE /author/3

Example Output:

    204 No Content
    201 No Content
    Status: OK
